Detailed Description

Core components of the event struct.

This interface is NOT ment to be used by the end-user to create an event. To use the event system you should inherit from the Event struct.

This interface should be used if, for some reason you need to store events of different types. Since the struct Event is templated, you cannot have them on the same list.
