Detailed Description

Dispatches the events and controls the Event System.

This class is used for dispatching the events. It is a core part of the Event System and should be on the main engine. The events will not work at all without this class.

To use it, just create it as a regular or unique pointer and call the PollEvents function when you want the events that exist on the queue to be dispatched. This is done this way to not disrupt the flow of the game.

This class is a Singleton, but cannot be used as such, instead using the SendEvent function will use the static instance of the class. Initializing an event system more than once will throw an exception
